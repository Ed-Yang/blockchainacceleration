=============================
Vitis Ethereum Mining Library
=============================

Vitis Ethereum Mining Library is an open-source Vitis Library written in C++ to mine coin of Ethereum using Xilinx Alveo cards. It provides:

* Source code to build acceleration kernel design, running on Xilinx Alveo Cards.
* Patch for **Ethminer**, to adopt and run acceleration kernels.
* Script to help build and run.
